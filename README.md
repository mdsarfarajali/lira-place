# 🏨 Lira Place - Palolem Beach Hostel

A modern, responsive website for Lira Place hostel in Palolem, Goa. Built with vanilla HTML, CSS, and JavaScript.

## 🌟 Features

- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Gallery** - Real property images showcasing rooms, facilities, and beach location
- **Pricing** - Clear pricing for Budget and Standard rooms
- **Contact Form** - Easy inquiry submission with WhatsApp integration
- **Location Map** - Embedded Google Maps showing proximity to Palolem Beach
- **Reviews** - Guest testimonials and ratings

## 📁 Project Structure

```
lira-place/
├── index.html          # Main website file
├── styles.css          # All styling
├── script.js           # Interactive features
├── room/               # Room photos
├── view/               # Beach and area views
└── other/              # Additional images
```

## 🚀 Getting Started

### Option 1: Open Directly
Simply open `index.html` in your browser.

### Option 2: Local Server
```bash
cd lira-place
python -m http.server 8000
# Visit http://localhost:8000
```

### Option 3: Live Server (VS Code)
1. Install "Live Server" extension
2. Right-click `index.html` → "Open with Live Server"

- Colors: Edit CSS variables in `styles.css` (`:root` section)
